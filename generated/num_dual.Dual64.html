num_dual.Dual64

class num_dual.Dual64(re, eps)

Dual number using 64-bit-floats as fields.

A dual number consists of f_0 + f_1 ε where f_0 is the function value and f_1 its first derivative.

Examples

>>> from num_dual import Dual64 as D64
>>> x = D64(1.0, 0.0)
>>> y = D64.from_re(2.0)
>>> x + y
3 + [0]ε

First derivative of a function.

>>> from num_dual import first_derivative
>>> import numpy as np
>>> f, df = first_derivative(lambda x: x * x + np.sqrt(x), 4.0)
>>> f
18.0
>>> df
8.25
